# AshokStudyHub 📘

A fullstack study material manager for uploading and viewing PDFs by semester & subject.

## 💻 Tech Stack

- Node.js + Express
- MongoDB (GridFS)
- Handlebars + Tailwind
- Multer + GridFSBucket

## 🚀 Features

- Upload PDFs by subject & semester
- View PDF files inline
- Admin login with upload control
